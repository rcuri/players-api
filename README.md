# Player API
An API written to retrieve player data stored in PostgreSQL database. PostgreSQL is hosted using AWS RDS.

### Stack
**Cloud**
- AWS Lambda
- AWS API Gateway
- AWS Cloudwatch
- AWS IAM

**IaC**
- Serverless Framework

**Python**
- runtime: 3.8
- [aws-psycopg2](https://pypi.org/project/aws-psycopg2/)
